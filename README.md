API REST desarrollada con Spring Boot para la gestión de tópicos de un foro, como parte del challenge del programa ONE de Oracle y Alura.

## Tecnologías utilizadas

- Java 17
- Spring Boot
- Spring Data JPA
- Spring Security
- JWT
- Flyway
- MySQL
- Maven
- Lombok

## Funcionalidades

- Registrar un tópico
- Listar todos los tópicos
- Ver un tópico por ID
- Actualizar un tópico
- Eliminar un tópico
- Autenticación con JWT
